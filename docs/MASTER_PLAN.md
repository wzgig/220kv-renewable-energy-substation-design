## 1. 总体目标

在任务书截止窗口内完成 220/35/10kV 新能源汇集变电所电气一次部分初步设计。设计必须同时满足工程逻辑、课程格式、计算可复核、图纸一致和公开仓库合规五项要求。

当前日期为 2026-07-15，任务书截止日期为 2026-07-17，因此采用“两日冲刺 + 后续完善”的安排：先锁定影响全局的方案和计算，再完成设备、图纸、文档与答辩；任何会导致大面积返工的绘图工作不得早于设备与净距冻结。

## 4. 分阶段执行细目

### P1 - 冻结任务与运行方式

1. 确认第 5 组题目和 220/35/10kV 系统范围。
2. 明确 220kV 两回在运、一回预留的本期与远期场景。
3. 明确 35kV 两段母线的馈线分配、母联运行方式和 N-1 限发策略。
4. 明确 10kV 系统是由主变第三绕组供电，还是由 35/10.5kV 辅助变压器供电；两种方案都要先做可行性核对。
5. 明确 0.4kV 所用电采用两台所用变、单母线分段、轮换暗备用；正常一台运行、母联闭合、另一台进线断开。
6. 将 110kV 通用负荷项标记为本题不适用，避免误计。
7. 决定主短路场景与敏感性场景：预留线投入/不投入、35kV 新能源贡献忽略/保守估算。

### P2 - 主接线与变压器方案

#### 220kV 侧

至少比较以下方案：

- 单母线分段：经济、清晰，但母线检修和故障影响范围较大；
- 双母线接线：运行灵活、检修便利，但设备多、占地和投资高；
- 必要时将带旁路或简化旁路作为课程比较项，而不是默认采用。

比较维度：回路数量、可靠性、检修方式、倒闸复杂度、扩建、占地、断路器数量和课程图纸可表达性。

#### 35kV 侧

以单母线分段的金属铠装开关柜方案为优先候选。两台主变分别接入两段，12 回本期集电线路和 2 回预留均衡分配；另在两段分别设置 `T10-1-HV`、`T10-2-HV` 电源变馈线间隔，设置母联并明确正常分列/并列方式。

#### 10kV 与 0.4kV

10kV 优先采用单母线分段，分别承载站用、无功补偿和辅助系统；0.4kV 按任务书采用三相四线制单母线分段，两台所用变轮换暗备用，正常一台运行并经闭合母联带两段。最终供电来源在 P1 冻结。

#### 变压器

- 计算每类 35kV 电源的视在功率并叠加 0.95 同时出力系数和 5% 线损。
- 校验 2×180MVA 的正常负载率、单台退出时的可送出比例和需要限发的容量。
- 对双绕组 220/35kV + 35/10.5kV 辅助变方案与三绕组主变方案进行结构、短路、运行和设备可得性比较。
- 所用变容量按全部经常负荷并留约 10% 裕度选择，短时负荷用于校验。

### P3 - 负荷与最大持续工作电流

统一使用以下计算链：

1. 单项视在功率：S_i = P_i / cosφ_i。
2. 35kV 综合汇集容量：各项 S_i 求和后乘 0.95，同时考虑 5% 线损。
3. 10kV 综合负荷：各项视在功率求和后乘 0.80，同时考虑 5% 线损。
4. 线路单回负荷按任务书给定回路数平均分配；计算书保留未取整值和最终设计值。
5. 三相回路电流：I = S / (sqrt(3) U)；最大持续工作电流所用系数必须说明来源。
6. 计算主变高、低压侧，220kV 线路，35kV 各类集电线路，35/10.5kV电源变馈线，10kV 各回路，母联和所用变回路电流；依赖未确认容量的职责必须显式保持待定。
7. 对 4% 年增长率进行远景敏感性说明，但不与本期设备额定值混写。

初步数量级校核：35kV 已装有功容量合计为 270MW；在逐项功率因数、0.95 同时出力和 5% 线损处理后，主变正常总负荷约处于 280MVA 量级。该值仅用于发现输入错误，正式结果以计算文件为准。

### P4 - 短路电流计算

1. 采用 100MVA 基准容量和 230/37/10.5kV 平均额定电压。
2. 系统电抗换算：X*_sys = x_c × S_B / S_i。
3. 线路电抗换算：X*_line = x × L × S_B / U_B^2。
4. 变压器电抗换算：X*_T = (u_k% / 100) × S_B / S_N；三绕组方案需先完成绕组电抗分解。
5. 至少设置 220kV 母线、35kV I/II 段及分段处、10kV I/II 段短路点；必要时增加设备端点。
6. 对最大运行方式计算三相短路初始对称电流、冲击电流和热效应所需量。
7. 主结果按新能源贡献忽略计算，保守场景按额定电流 1.1～1.2 倍补充，避免设备裕度判断偏乐观。
8. 用独立复算表或脚本进行一次基准值、串并联和单位检查。

### P5 - 设备选择与校验

每类设备至少核对：

- U_e >= U_n；
- I_e >= I_max；
- 断路器额定开断电流不小于计算短路电流；
- 额定峰值耐受电流不小于冲击电流；
- 短时耐受电流与持续时间满足热稳定；
- 环境温度、安装方式、海拔（任务书未给时不得擅自假定高海拔）和污秽条件适配。

设备范围：220kV SF6 断路器与隔离开关、35kV 真空/SF6 开关柜、10kV 真空开关柜、母线/导体、绝缘子和穿墙套管、CT、PT、避雷器、主变与辅助/所用变。

母线还需校验长期载流、温度修正、热稳定、动稳定、机械强度；220kV 导体补充电晕校验。所有型号必须有可追溯的厂家样本或可靠产品数据。

### P6 - 配电装置设计

- 220kV：比较户外 AIS 与 GIS。课程断面/平面表达优先考虑户外 AIS，但最终以可靠性、占地、气候、设备尺寸和教师要求为准。
- 35kV：优先室内金属铠装移开式开关柜，形成两段母线、母联、主变进线、12 回本期馈线、2 回T10电源变馈线和 2 回预留的间隔表。
- 10kV：室内开关柜，按站用、无功补偿、冷却及备用电源组织间隔。
- 断面与平面必须使用同一设备外形尺寸、安全净距、间隔宽度、道路和出线方向参数。

### P7 - CAD 图纸

1. 建立 GB 图框、图层、线型、线宽、文字、标注和设备编号规范。
2. 建立断路器、隔离开关、CT、PT、避雷器、变压器、母线和接地符号块库。
3. 用结构化拓扑生成主接线简图，人工复核回路和设备配置。
4. 设备与净距冻结后绘制典型间隔断面图。
5. 按出线方向、间隔顺序、道路和检修要求绘制平面布置图。
6. 通过 accoreconsole 批量检查图层、范围、字体、打印和格式。
7. 对最终 PDF/PNG 完成逐图目检和打印范围复核。
8. 交付可再生 DXF/数据、PDF 和 PNG 预览；本地保留可编辑的 2018 兼容 DWG。

### P8 - 说明书、计算书与答辩

说明书按任务书建议目录组织：原始资料、主接线、负荷与变压器、持续电流与短路结果、设备选择、配电装置、设备总表。计算书保留完整代入、单位和中间结果。图纸以附录或独立图册提交。

答辩材料至少覆盖：为什么选该主接线、2×180MVA 是否合理、N-1 如何处理、短路点为何这样选、断路器开断与动热稳定如何通过、35kV 如何分段、三张图如何保持一致。
