# 项目执行规范

## 重构后的工作提示词

> 以“完成第5组220kV新能源汇集变电所电气一次部分课程设计，并形成可手绘临摹、可复核、可持续维护的完整工程底稿”为总目标。完整使用本地任务书、课程资料和已冻结计算；CAD图作为正式工程底图，用户最终按图手绘临摹。参数采用三级证据优先级：第一，任务书和教师口径；第二，现行国家/行业标准及官方厂家资料；第三，任务书确实缺失时采用偏保守、适合课程设计的工程假设。所有第三类取值必须写明“课程设计假设”、选择理由、适用条件和对结果的影响，但不因缺少非关键现场数据而无限期停留在pending。设计计算、设备职责、主接线、平面图、断面图、说明书和计算书必须使用同一数据源；每个有效批次完成再生成、测试、CAD/PDF目检、日志、提交、推送和GitHub Actions核验。只有会改变根本拓扑、造成明显安全风险或无法合理保守取值的问题才再次向用户提问。

完整、可直接复用的结构化版本见 [课程设计执行提示词](EXECUTION_PROMPT.md)，全部题目给定值、标准采用值和课程设计假设见 [设计依据与假设台账](DESIGN_ASSUMPTIONS_REGISTER.md)。

## 本轮用户授权

- CAD图可以完整绘制并作为手绘临摹依据；最终提交形式仍按课程要求由用户手绘。
- 任务书未给出的海拔、污秽、保护时间、无功容量、系统运行许可等参数，可以通过现行标准、官方资料和课程设计计算自行拟定。
- 拟定值不是现场施工设计参数；若后续获得教师或真实场址数据，必须由统一输入文件触发计算、图纸和文档同步更新。

## 成功标准

项目只有在以下条件同时满足时才算完成：

1. 任务书中的每项内容都有明确产物，并能在需求追踪矩阵中定位。
2. 说明书、计算书、设备表和三张图使用同一组参数，不互相矛盾。
3. 每个设备选择均有额定电压、持续电流、开断、动稳定和热稳定依据。
4. CAD 图纸可编辑、可打印、字体不缺失，并导出兼容版 DWG 与 PDF。
5. 原始资料和公开成果严格分离，Git 历史中不出现隐私或第三方受限文件。
6. 每个有效修改批次完成检查、日志、提交、推送和远端验证。
7. 任务书缺失参数形成独立假设台账，能区分“题目给定”“标准采用”“课程假设”和“最终待教师复核”。

## 工作边界

- 不把参考设计书直接改名为本项目成果。
- 不从旧参考图按比例猜尺寸；缺失尺寸必须来自现行标准、官方厂家外形图或明确标注的课程设计假设。
- 不把课程设计假设伪装成任务书原始条件或现场施工定值。
- 不把 CAD/Office 自动保存文件、提交压缩包或本地环境写入仓库。
