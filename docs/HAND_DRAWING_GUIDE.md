# 四张CAD图纸手绘临摹指南（三张必交图 + 一张增强详图）

## 通用顺序

1. 先画A1图框、标题栏、图名、图号和比例。
2. 用2H铅笔轻画母线、设备中心线、道路/构架和主要尺寸定位线。
3. 按“母线/导线→设备→编号→尺寸→说明”的顺序描深；母线最粗，一次导线次之，尺寸与文字最细。
4. 预留对象使用虚线；正常断开的断路器必须画出开口状态。
5. 最后核对四图中的L1/L2/L3、T1/T2、T10-1/T10-2、SVG-1/SVG-2、MOA、ES及母联状态是否一致。

## SLD-01 电气主接线简图

- 从上到下依次画220kV、35kV、10kV、0.4kV四层母线。
- 220kV分段断路器、35/10kV母联正常断开，0.4kV母联正常闭合。
- 220kV：L1、L2在运，L3预留；两台180MVA、220/35kV、YNd11、uk=14%主变分别接两段。
- L1/L2旁同时保留“正常367.780A”和“N-1需735.559A（须限发）”，不得只抄后一个数值。
- 35kV：12回集电线路按6+6分配；WA-1～4、WB-1～3标“架空+入口MOA”，PVA-1～2、PVB-1～2、ES-1标“电缆+ZCT”，另有R1/R2和T10-1-HV/T10-2-HV。
- 10kV：T10为2×31.5MVA、35/10.5kV、YNd11、uk=8%；每段各接1套±12Mvar SVG。
- 10kV三类辅助负荷各画1回：站用电备用电源、无功补偿及冷却、集控通信及监控；不要把三类负荷在两段各重复画一遍。两回SVG和三类电缆辅助馈线标“ZCT”。
- 35kV各段按“1000kVA ZN接地变+低电阻”画出并标400A/10s、R≈50.5Ω；10kV各段按“200kVA ZN接地变+低电阻”标200A/10s、R≈28.9Ω。容量和10s短时过载均属课程假设，最终由接地故障、保护配合和厂家成套资料覆盖。
- 在运行说明中抄写：正常分列时每段1套接地源；35/10kV母联合闸前退出受电/故障段接地源，只保留健康侧1套；禁止母联闭合且两套接地源并联；重新分列后恢复每段1套。
- 220kV线路入口与主变端配置YH10W-204/532；35kV母线PT柜/主变端配置YH5WZ-51/134；10kV母线PT柜及T10/SVG端配置YH5WZ-17/45。
- 220kV线路侧隔离开关的线路一侧画常开ES，不得把ES支路接在QS母线侧；35/10kV柜内ES按常开表达并注明机械/电气联锁。
- 0.4kV：TS1运行、TS2暗备用，所用变为2×200kVA、SCB14、Dyn11、uk=4%，母联闭合；两进线不得同时闭合。

## LAY-220-01 平面布置图

- 比例1:200，北向上；先画145m×90m站区边界、围栏和4m检修道路。
- 三相母线按4.0m相间中心距画三条平行线；I、II段中间为正常断开的分段间隔。
- 典型功能中心按14m节距组织：L1、T1、PT-I、分段、PT-II、T2、L2、L3预留。
- 平断面纵向尺寸同源：母线至线路构架29m、至主变构架20m、至主变中心27m；临摹时不要分别估算。
- L1/L2/L3均向西北引出，分别标70km、85km、60km和LGJ-400/50。
- 图中道路、建筑和设备外形为课程设计占位，不要删去“精确厂家图覆盖”的说明。

## SEC-220-01 典型间隔断面图

- 比例1:100；左侧画线路间隔A-A，右侧画主变间隔B-B。
- 线路A-A设备顺序：管形母线→母线侧隔离开关→SF6断路器→CT→线路侧隔离开关→CVT→避雷器→出线构架。
- 主变B-B设备顺序：管形母线→母线侧隔离开关→SF6断路器→CT→避雷器→主变构架→180MVA主变。
- 标高：母线+9.0m、线路构架+14.5m、主变构架+11.5m；相间距4.0m。
- 净距表照抄：A1=1800、A2=2000、B1=2550、B2=1900、C=4300、D=3800mm。

## SEC-220-L1-01 L1线路间隔增强详图

- A1横向、比例1:50，按“220kV I段管形母线→母线侧QS→QF→TA→线路侧QS+常开ES→CVT→YH10W-204/532 MOA→出线构架”从左到右临摹。
- 补齐设备编号、额定等级表、基础轮廓、构架与设备接地线、地下接地网接口、水平尺寸、主要标高以及A1/A2/C净距。
- 地下接地网只画接口和示意，不填写臆测的接地电阻、接触电压或跨步电压；没有雷暴日、建筑高度和保护模型时，不声称避雷针保护范围已完成。
- 设备外形和基础仍为课程占位，临摹时保留“最终厂家图覆盖”说明。
