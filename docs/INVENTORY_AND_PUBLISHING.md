# 资料盘点与公开发布边界

## 盘点结论

初始化时共发现 128 个文件、13 个目录，总量约 283MiB。资料覆盖任务书、模板、课程讲义、学生指导书、设备手册、参考设计书、手稿照片、17 个参考 DWG，以及本组已有的接线、断路器和母线资料截图。

关键技术来源已经读取并归纳：

- 本组任务书：8 页，确定题目、系统参数、负荷、统一计算原则、交付物和建议目录。
- 课程设计安排：两周，第一周以主接线、负荷、主变和短路计算为主，第二周完成设备选择、文档、图纸和答辩。
- 学校模板：规定封面、摘要、目录、章节、公式、表格、图题和参考文献位置。
- 课程资料：覆盖主接线、厂用电、导体与设备选择、配电装置等理论。
- 参考 DWG：可帮助理解图框、间隔和布置，但存在版权、字体缺失和二进制历史膨胀风险。

## 公开风险

| 风险 | 发现 | 处理 |
| --- | --- | --- |
| 明确限制传播 | 存在文件名直接标注不可传播的资料 | 永不进入公开历史 |
| 第三方版权 | 课程讲义、教材手册、他人设计书和工程 DWG | 仅本地参考 |
| 个人信息 | 姓名、班级、教室、课程时间、作者和修改者元数据 | 公开文档重新编写并脱敏 |
| 精确位置与设备信息 | 图片含 GPS、手机型号和拍摄信息 | 原图不公开；必要预览先清除元数据 |
| 团队成果权属 | 本组目录可能包含共同成果 | 未获组员同意前不公开原件 |
| 仓库体积与二进制元数据 | 多个 PDF/DWG，且 DWG 保存时写入机器登录信息 | 第三方文件排除；原创 DWG 本地生成，公开 DXF/PDF/PNG |

## 允许清单

公开仓库仅允许以下内容：

- 脱敏后的项目范围、设计输入、计划、决策和追踪矩阵；
- 自行完成的计算过程、结果表和校验脚本；
- 自行绘制的 CAD 源图、标准、块库和 PDF/PNG 预览；
- 自行撰写的说明书、计算书、总结和答辩材料；
- 项目健康检查、生成和转换脚本；
- 不包含个人信息的最终交付说明。

## 发布前检查

1. 检查 Git 跟踪清单，不使用无审查的 git add .。
2. 检查文档作者、最后修改者、自定义属性和修订记录。
3. 检查图片 EXIF、DWG 摘要信息、外部参照绝对路径和字体依赖。
4. 检查文件名是否包含姓名、学号、班级或限制传播提示。
5. 检查 DWG 未被公开跟踪，压缩包和媒体已被忽略。
6. 运行 scripts/project_health_check.ps1 后再提交。
