# 设计依据与假设台账

## 1. 分类规则

| 等级 | 含义 | 使用规则 |
| --- | --- | --- |
| A | 任务书或教师给定 | 不得被网络资料覆盖 |
| B | 现行标准、官方资料或本课程正式课件采用 | 记录版本、页码/链接和适用条件 |
| C | 课程设计假设 | 明确理由、保守性、影响和替换触发条件 |
| D | 最终待教师/真实场址复核 | 不阻塞课程底稿，但不得冒充施工设计条件 |
