# 短路电流计算模块

该模块按任务书给定的100MVA标幺制、平均额定电压和X-only简化口径，完成当前已知条件下的220kV、35kV三相短路初算。GB/T 15544.1-2023作为最终复核依据，但本模块当前不是该标准的完整实现。

运行：

~~~powershell
python -m calculations.short_circuit.calculate
python -m unittest discover -s tests -v
~~~

当前计算点：

- 220kV母线分段断路器闭合的条件性最大基准方式，等待两系统并列许可确认；
- 220kV两段分列的正常可行方式；
- 35kV保持两段分列，并分别计算220kV母联闭合与分列两种上游方式；
- 35kV 两台健康主变低压侧并列的非正常敏感性方式；
- 35kV 新能源贡献按额定电流 1.1～1.2 倍的保守范围。

10kV 结果尚不定稿，因为两台 35/10.5kV 电源变的容量取决于尚未给出的无功补偿 Mvar、启动和备用要求。

初算按任务书口径忽略电阻，并把k=1.8作为可配置的课程假设估算电网峰值。该k值不是由“忽略电阻”自动推导；设备校验前必须结合课程要求、R/X、标准电压系数、变压器修正系数和峰值系数κ重新核对。

输出中的峰值只包含电网贡献。35kV新能源贡献目前只以RMS算术上界列示，220kV故障点的新能源贡献和新能源峰值尚未建模，因此不得直接把当前峰值作为动稳定最终结论。第三回220kV线路L3不参加本期基准计算，远期投入场景另算。
