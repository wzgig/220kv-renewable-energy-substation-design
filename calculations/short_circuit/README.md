# 短路电流计算模块

该模块按任务书给定的100MVA标幺制、平均额定电压和X-only简化口径，完成220kV、35kV和10kV三相短路课程初算。GB/T 15544.1-2023作为最终复核依据，但本模块当前不是该标准的完整实现。

运行：

~~~powershell
python -m calculations.short_circuit.calculate
python -m unittest discover -s tests -v
~~~

当前计算点：

- 220kV分段断路器闭合的条件性最大方式，等待两系统并列许可确认；
- 220kV两段分列的正常可行方式；
- 35kV保持两段分列，并分别计算220kV分段断路器闭合与分列两种上游方式；
- 35kV 两台健康主变低压侧并列的非正常敏感性方式；
- 10kV两段分别由单台31.5MVA T10供电，并计算上游220kV分列/条件闭合方式；
- 两台健康T10经10kV母联并列的禁止敏感性方式；
- 220/35kV新能源贡献和10kV SVG贡献按额定电流1.1～1.2倍形成RMS课程上界。

10kV课程输入已冻结为2×31.5MVA、35/10.5kV、uk=8%的T10和2×±12Mvar SVG，因此10kV分列与禁止并列敏感性均已纳入可再生结果。

初算按任务书口径忽略电阻，并把k=1.8作为可配置的课程假设作用于电网和变流器RMS算术上界，形成等级预筛用的课程综合峰值。该k值不是由“忽略电阻”自动推导；设备最终校验必须结合R/X、标准电压系数、变压器修正系数、峰值系数κ和厂家变流器动态模型重新核对。

当前控制职责包括：220kV分列4.033kA/10.266kA、分段条件闭合7.385kA/18.798kA；35kV条件性上限16.291kA/41.470kA；10kV条件性上限15.638kA/39.808kA（RMS/课程综合峰值）。输出中的`grid_peak_current_ka`仍只表示电网分量，不能与课程综合峰值混用。新能源和SVG的精确动态、衰减、相位、R/X及标准峰值模型尚未建立，因此不得把课程峰值写成最终动稳定结论。第三回220kV线路L3不参加本期基准计算，远期投入场景另算。
