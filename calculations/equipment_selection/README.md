# 主要电气设备预筛选模块

该模块把设备选择拆成三个层次：

1. 从现有负荷、持续电流、短路结果和主接线基线生成回路职责；
2. 用 `data/equipment_selection.yaml` 管理场景、回路组和候选分配；
3. 用 `data/equipment_catalog.yaml` 管理候选额定值及证据状态。

运行：

~~~powershell
python -m calculations.load_and_transformers.calculate
python -m calculations.short_circuit.calculate
python -m calculations.equipment_selection.calculate
python -m unittest discover -s tests -v
~~~

当前输出同时包含“开关设备额定值等级预筛选”和“课程设计专项闭环”，但不把课程目标表伪称为厂家订货型号。任何最终工程缺失值都会传播为 `pending`，不会被当成零、跳过或自动通过。

当前可以自动完成：

- 220kV线路N-1、主变高低压侧和35kV各类馈线持续电流职责；
- 220kV分段按735.559A保守持续职责闭环；
- 35kV两段按实际馈线分配的电流复核；
- 220kV、35kV、10kV必选、条件性和禁止运行方式的短路场景分级；
- 候选额定电流、开断电流、关合/峰值和短时耐受电流的算术预筛；
- 220kV Φ100/90铝合金管形母线、35kV每相3×125×10及10kV每相2×125×10铝母线的载流与C=87、t=1.10s热稳定校核；
- 220kV管形母线的简化动稳定与Peek公式电晕预校核；
- 220/35/10kV金属氧化物避雷器持续运行电压和LIWV/残压保护裕度校核；
- 35kV架空入口MOA与实际架空馈线组的接口映射，以及35/10kV电缆馈线ZCT覆盖与限定接地电流目标校核；
- CT变比、准确级目标及对应回路动热稳定预校核，以及PT、绝缘子/套管、接地开关课程选择表；套管补齐Um、LIWV、持续电流及动热稳定；
- 35kV每段一套ZN接地变+低电阻设备包（400A/10s、约50.5Ω、约8.083MVA、最小约808kVA、选1000kVA）和10kV每段一套设备包（200A/10s、约28.9Ω、约1.155MVA、最小约115.5kVA、选200kVA）；
- 接地变馈线开关柜、相/中性CT课程目标、保护功能及“母联合闸前退出受电/故障段接地源、禁止两套并联、分段恢复后各段一套”的联锁逻辑；
- 35/10kV短路水平低于31.5kA开关设备能力，课程方案不设限流电抗器；高压熔断器仅保留为PT柜厂家配置项。

当前必须保持待定：

- 厂家最终绝缘水平、外绝缘爬距、机械负荷、海拔和污秽适配；
- 最终保护配合、断路器开断数据及标准化短路热效应；
- 新能源和SVG的精确故障电流及动态峰值模型；
- 避雷器接地方式、能量耐受和完整绝缘配合；
- CT/PT负担、饱和、暂态级、铁磁谐振及二次回路最终分配；
- 母线支撑共振、端部受力、厂家温升和连接金具；
- 35/10kV矩形母线的支撑几何、共振、连接受力和机械动稳定（当前明确为 `pending`，不得视为已校核通过）；
- 接地变零序阻抗、接地电阻10s温升、相/中性CT饱和、保护定值、硬接点/软件联锁和完整厂家设备包；
- ZCT窗口尺寸、电缆屏蔽层接地回流路径、负担、拐点与精确型号；
- 厂家精确型号、当前样本、外形尺寸和室内40℃/室外41℃载流确认；
- 接地开关感应电流开合等级，以及PT柜熔断器的厂家配合与选择性。
