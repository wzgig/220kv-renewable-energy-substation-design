# 本地私有材料区

此目录保存课程任务书原件、模板、课程讲义、教材/手册、他人设计书、参考图纸、扫描手稿和其他仅供本地核对的资料。

除本说明文件外，目录内容全部由 .gitignore 排除，原因包括：

- 含第三方版权材料或明确限制传播的文件；
- 含真实姓名、班级、教室、时间、GPS、设备型号或文档作者元数据；
- 含组员共同成果，公开权属尚未确认；
- 大型 PDF/DWG 会显著膨胀仓库历史；
- 云盘、Office 和 CAD 临时文件没有版本价值。

公开成果应从这里提取必要技术参数后重新编写，并在发布前完成脱敏、版权和元数据检查。不得通过强制添加将本目录原件提交到公开仓库。
