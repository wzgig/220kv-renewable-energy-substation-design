## 再生成顺序

DOCX 每次重大修改后均需渲染为页面图片逐页检查，再输出最终 PDF。如果必须重新生成报告，顺序应为：

1. 生成 DOCX。
2. 在 Word 中更新目录并导出 PDF。
3. 对最终 DOCX/PDF 执行隐私和元数据清理。
4. 逐页渲染复核，确认页数、图表和数值未漂移。

> [!WARNING]
> 已完成隐私清理的 DOCX 不应再用 Word 保存，否则可能重新写入本机的最后修改者信息。
