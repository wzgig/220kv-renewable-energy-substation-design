# 说明书与计算书补充重构计划

## 任务目标

将现有《技术设计说明书》和《技术设计计算书》从以结果表格为主的生成稿，扩充为具有完整设计论证、公式推导、数值代入、结果校核和工程边界说明的正式课程设计报告。

## 交付物

- 更新后的 `report/01_220kV新能源汇集变电所_技术设计说明书.docx/.pdf`
- 更新后的 `report/02_220kV新能源汇集变电所_技术设计计算书.docx/.pdf`
- 同步更新的报告生成脚本、报告说明、项目日志和验证期望
- 修改前 DOCX/PDF 的本地备份，位于忽略目录 `tmp/original_reports_20260718/`

## 内容约束

- 以 `data/*.yaml`、`calculations/results/**`、`docs/DESIGN_ASSUMPTIONS_REGISTER.md` 和既有设计决策为唯一数值来源。
- 不新增无法追溯的设备型号、标准数值、现场条件或保护定值。
- 每个关键环节至少包含方法说明、公式、符号和单位、代入、结果及工程解释。
- 保留“课程预筛”和“最终工程待复核”的边界，不把缺失输入按零处理或自动判为通过。
- 说明书突出方案与工程含义；计算书突出完整计算链，避免两份文档机械重复。

## 重点补充范围

1. 主接线方案比较、各电压级接线和事故转供逻辑。
2. 35kV/10kV负荷、主变、所用变、T10和SVG的容量计算。
3. 各回路持续工作电流、母线段多口径说明和LGJ-400/50温度修正。
4. 标幺基准、系统/线路/变压器电抗、220/35/10kV短路点及变流器贡献。
5. 设备额定电压、电流、开断、峰值与热稳定I²t校验。
6. 母线截面与热稳定、220kV管母简化弯曲和电晕。
7. 35/10kV接地电阻及接地变容量、MOA绝缘裕度和联锁边界。
